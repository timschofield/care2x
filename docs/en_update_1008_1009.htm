Updating (beta 1.0.08) to (beta 1.0.09)

NOTE: The following update procedure is valid ONLY if you have a working 1.0.08 installation.

1. Uncompress the CARE 2002 beta 1.0.8 scripts into a temporary directory.

2. Copy all the files EXCEPT the /include/inc_init_main.php file into the directory where Care is installed.

Tip: If you have a file manager that compares file dates, use it to copy only the modified files.
3. Start your phpmyadmin (or any other mysql admin program).

4. Run the following sql script:

install/sql/db_update_1007_1008.sql
5. Convert the existing user passwords to md5 encrypted form by typing http://host_address/install/encode_pw_md5.php into your browser.
Enter your previous admin username and password and hit "Yes, proceed" button.

6. After the conversion of passwords, REMOVE the following files from Care directories:

  • /create_admin.php
  • /install/encode_pw_md5.php
  • /install/initialize.php

    Tip: You can move them to other place outside the web server directory.

    Note: You must convert your existing user passwords as described at step #6 BEFORE adding new users to Care2x.

    Note: I have used the sql script to update the online demo . It worked well. However, it was not an exhaustive test, so please notify me at once if any problem occurs. Thanks. Elpidio
