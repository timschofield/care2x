The following are new in the Care2x beta 1.0.09 release:

A. Passwords are encrypted with MD5

  • The user passwords are now stored in the database in encrypted form.
  • A script is supplied for creating system admin users.
  • Another script is supplied for encypting existing user passwords from the previous version to version 1.0.09.
    • B. Global session time-out

    • A global session time-out for password protected modules is now implemented. This time out will lock the module after a configured length of time without user activity has elapsed. The user can only resume work after refreshing the browser and logging in anew.
    • The session time out can be disabled or enabled by the system administrator.
    • The length of elapsed time can be configured by the system administrator.
    • This time out is one of several requirements for HIPAA compliance.

    C. The system administration module was redesigned for better navigation.

    • The submenu items are now listed on an additional index frame.

    D. Additional OP rooms can now be created.

    • Creating new OP rooms can be done with a GUI page in the system admin module.
    • Each OP room can be given a name or short description.
    • Each OP room can be closed or opened via the system admin module.

    E. GUI improvements.

    • The "Quick info" that appears on the first page can now be edited via a GUI page. Only the system administrator can edit the information via the sytem admin module.
    • The maximum number of rows displayed on each page during search actions can now be configured by the system administrator via a GUI page.
    • Several attributes of the news display like title, preface, news body can now be configured for width, color, font and size. The configuration can be done via a GUI page in the system admin module.

    F. New languages were added.

    • Arabic started by Walid Fathalla (LY)
    • Chinese-simplified started by Wei Ning Chen (CH)
    • Spanish and Brazilian-Portuguese versions are almost complete.

    G. Some changes in terminologies used.

    • OP logbook is now called OP journal.
    • OR is now called OP Room.
    • EDP is now called System Admin

    H. Much more stable than previous versions

  • Lots of debugs done and patches made.
  • The filename for the person's uploaded id picture is now named after his PID number to avoid problems with special characters.
  • Most active bug reporter was Thomas Wiedmann (DE).
    • I. Help pages were enhanced and expanded.
  • Several technical and administration manuals were created and available at http://care2x.net/docswiki/index.php?FrontPage
  • The integrated phpMyAdmin was replaced with a version 2.5.3 release.
